# Gross Pay Calculator with Overtime

A simple Python program that calculates an employee's gross pay based on hours worked and hourly rate. The program applies the standard hourly rate for the first 40 hours and an overtime rate of 1.5× the hourly rate for any additional hours worked.

## Features

* Accepts user input for username, hours worked, and hourly pay rate.
* Calculates regular pay for up to 40 working hours.
* Applies overtime pay at 1.5× the hourly rate for hours worked beyond 40.
* Displays the calculated gross pay along with the user's name.

## How to Run

1. Make sure Python 3 is installed.
2. Run the script:

   ```bash
   python gross_pay.py
   ```
3. Enter your username, hours worked, and hourly rate when prompted.
